# Todo List

This is a Todo List application where you can manage your tasks. You can add todos with a description and set their priority. Once added, you can view them in the list.

## Features

- Add todos with a description and priority
- View todo details
- Delete todos
- Mark todos as completed by clicking on them
- **Drag & Drop todos** to reorder them
- Filter todos by:
  - Name
  - Status (completed, not completed, or all)
  - Priority
- Sort todos by:
  - Creation date
  - Priority  
    Both ascending and descending order are supported

## Technologies Used

- **Zod** for input validation
- **Redux** for state management
- **Tailwind CSS** for styling
- **LocalStorage** for storing data
- **TypeScript** for type safety
- **React Query** is installed but not used yet; it was intended for handling API requests, but for now, the app uses ready-made APIs.
- **Vite** for project setup

## Getting Started

```bash

2.Install dependencies:
```
